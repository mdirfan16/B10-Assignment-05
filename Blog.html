Important FAQ?

What is the DOM?

The DOM is a programming interface for web documents, representing the structure of a webpage

How to create a new element in the DOM?

Use document.createElement('tagName').

How to select an element by its ID?

Use document.getElementById('elementId').

How to remove an element from the DOM?

Use element.remove();

How to change an element's style?

Use element.style.property = 'value';
